#EutheriaWiki Agent Guide
The wiki lives in wiki/.
Update documents live in updates/inbox/.
Processed update documents may be moved to updates/processed/.
Update existing articles when possible.
Create new articles when a topic is substantial and distinct.
Add cross-links between related articles.
Update category index pages when new articles are created.
Do not invent unsupported facts.
Prefer detailed, encyclopedic Markdown that remains readable and well-structured.
Avoid duplicate articles for the same topic.
Interpret the instruction "Update Wiki" to mean: process all update files in updates/inbox/, applying each file's content to the wiki by editing existing articles and creating new ones as needed.
When canon supports it, expand entries with fuller historical, political, religious, and geographic context rather than minimal summaries.
